# CommitPot パフォーマンス実機チェックリスト

## 対象ビルド
- iOS: TestFlight 最新ビルド
- Android: Closed testing 最新ビルド

## 事前条件
- アプリを完全終了してから起動
- 通信が安定した環境で実施
- 端末の省電力モードはOFF

## 判定基準
- タップから視覚反応まで 100ms 前後で反応する
- 「無反応（2回以上押してやっと反応）」が 10 回中 0 回
- 300ms 以上の体感フリーズがない

## シナリオA: ダッシュボード
1. `新しく鍋を作る` を10回連続で開閉
2. 各カードの `SHOP` を10回連続で開閉
3. タブ (`HOME/SHOP/STATS`) を往復で10回
4. 戻る矢印を連続5回

### 記録
- 反応しなかった回数
- 明確に遅いと感じた回数

## シナリオB: Builder
1. タンパク質を連続で切り替え（最低10回）
2. 2つ目選択時の推薦表示を確認
3. 野菜を複数回変更（AI推薦→カスタム→再推薦）
4. `次へ`/`戻る` を往復10回

### 記録
- 古い推薦結果が後から反映される事象がないこと
- HUD/Toast表示中でもタップ阻害がないこと

## シナリオC: Shopping/Cook
1. 買い物リスト画面を開く
2. 画面遷移直後にスクロール・戻る・ボタン操作
3. 仕込みモード遷移→戻るを5回

### 記録
- 遷移直後に固まる感じがないこと
- ボタン押下が取りこぼされないこと
